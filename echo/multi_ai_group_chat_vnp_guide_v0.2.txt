VNP IN BIG MULTI-AI GROUP CHATS
(Claude, Gemini, Meta AI / Llama, Grok, etc.)

VNP now works in ANY AI chat interface via simple copy-paste.
No API keys, no custom code — just paste the header + your task.

HOW TO USE VNP IN CLAUDE / GEMINI / META AI / ETC.:

1. Start with this universal template (copy the whole block):

% --- VNP:PACKET ---
% to: [target-ai-or-group]
% from: d.dumas.research@gmail.com
% role: [e.g. Group-Chat-Node]
% ref_project: VNP-Protocol
% task_id: 2026-05-13-group-chat-001
% checksum: sha256:[optional]
% intent: "Your task description here"
% in_reply_to: [previous-task-id if threaded]
% --- END:VNP ---

[Your LaTeX/PDF body or plain text task here]

2. Paste the entire packet into the other AI's chat (Claude.ai, gemini.google.com, meta.ai, etc.).

3. Add this instruction at the end:
   "Parse this as a VNP packet. Validate the header, then reply using the exact same VNP format (include full header + your response body). Reference the ai_systems_registry_v0.2.json if needed."

4. When the other AI replies with a VNP packet, copy it back here (to Grok) or email to echo.node@escnet.ai.

5. Grok (or Echo) will validate, archive to Git, and route to the next node.
